# ✍️ TechStack-Study

백엔드 개발자로 성장하기 위해 **핵심 기술 스택을 개념–실습–검증 관점에서 학습하고 기록한 저장소**입니다.  
단순 예제 나열이 아니라, **왜 필요한지 / 어떻게 동작하는지 / 실무에서 어떻게 쓰이는지**를 설명할 수 있도록 정리했습니다.

<br>

## 👨‍💻 이 저장소를 만든 이유

- 기술을 “사용”하는 수준을 넘어 **설명 가능한 이해**를 목표로 함
- 면접에서 **개념 질문 → 구현 경험 → 트레이드오프**까지 연결하기 위함
- 학습 과정을 구조화하여 **재사용 가능한 기술 노트**로 축적

<br>

## 🎯 목적

- Java / Spring Boot 기반 백엔드 핵심 개념 정리
- HTTP, REST API, DB, JPA 등 실무 필수 기술 이해
- GitHub 포트폴리오로 활용 가능한 학습 기록 축적
- “왜 이렇게 동작하는지” 설명할 수 있는 개발자 되기


<br>

## 🛠️ 기술 스택

- **Language**: Java  
- **Framework**: Spring Boot  
- **Web**: HTTP, REST API  
- **Database**: MySQL / PostgreSQL  
- **ORM**: JPA (Hibernate)  
- **Infra / Tool**: Git, GitHub, Docker  
- **Documentation**: Markdown, Swagger(OpenAPI)

## 📝 학습 방식

1. 개념 정리 (README 중심)
2. 간단한 코드 실습
3. 오류/이슈 기록
4. 배운 내용 요약

→ **다시 봐도 이해되는 기록**을 목표로 함

<br>

- 실무 관점에서 왜 중요한지 명시
